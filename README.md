My project is a full-stack web application for a bakery called ChocoDelicias. 
It allows users to browse products, log in, and interact with a real-time chat system. 
It also includes an admin panel to manage products.
First, I defined the idea of the application, which is a bakery web app with user and admin functionalities.
I decided the features such as login, product management, and chat system.
I planned the project, built the frontend using React, developed the backend with Node.js and Express, connected MongoDB, implemented authentication using JWT and bcrypt, created an admin panel, tested the application, uploaded it to GitHub, and finally deployed it online.
